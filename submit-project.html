Submit a Project

Have a small, practical project that could benefit from shared effort? Send the details and a few photos so it can be reviewed for fit, safety, and scope.

Project submission does not guarantee selection. Each project is reviewed before being considered for a future build day or volunteer work session.

Good Fit for This Project

  • Cleanup and prep work
  • Painting and refresh work
  • Yard and exterior cleanup
  • Organizing neglected spaces
  • Simple, beginner-friendly repair tasks
  • Small improvement projects with visible impact

Outside Current Scope

  • Major structural work
  • Large-scale renovations
  • Major electrical, plumbing, or HVAC work
  • Hazardous or unsafe environments
  • Projects requiring licensed professionals
